Pure Function 

1. Первое правило, которому должна придерживаться чистая функция - имьютабельность.
Функция не должна мутировать(изменять) данные, которые в нее приходят. 
Должна только использовать входящие данные и выдавать на основе их результат.

reducer - Тоже является чистой функцией, мы принимаем данные из вне(state и action например) и создаем копию state не изменяя его напрямую

2. Чистая функция обязательно имеет return и возвращает нам что-то.

3. Если же мы хотим что-то изменить в нашей чистой функции, то обязаны создать копию.

4. Функция не должна иметь side-эффектов. 
То есть не менять входящие данные, не использовать глобальные переменные, не использовать запросы на сервер (Не должна иметь асинхронных запросов).

5. Детерминированость, идемпотентность - Значит, что внезасимисоти от входящих данных функция вернет один и тот же результат.
